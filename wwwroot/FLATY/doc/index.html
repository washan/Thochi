FLATY comes with a huge collection of plugins and UI components and it works in all major web browsers, tablets and phones.

3. Theme Configuration

To top

Setup Theme

FLATY comes with 9 color themes, blue or nothing, blue is default skin color, red, green, orange, yellow , pink, magenta, gray, black. To setup a selected theme, add skin-[theme_name] class in body's tag. For instance, if you like to use pink theme, add skin-pink class in body's tag, like below: 

...
<!-- Page with default pink skin for navbar, sidebar and boxes -->
<body class="skin-pink">
...

Navbar Color

FLATY comes with 9 color for navbar, blue or nothing, blue is default navbar color, red, green, orange, yellow , pink, magenta, gray, black. To setup a selected color, add navbar-[color_name] class in #navbar's tag. For instance, if you like to use pink color, add navbar-pink class in #navbar's tag, like below: 

<!-- BEGIN Navbar -->
<div id="navbar" class="navbar navbar-pink">
    ...
</div>
<!-- END Navbar -->

Consider that navbar color, will overrides skin color.

Sidebar Color

FLATY comes with 9 color for sidebar, blue or nothing, blue is default sidebar color, red, green, orange, yellow , pink, magenta, gray, black. To setup a selected color, add sidebar-[color_name] class in #main-container's tag. For instance, if you like to use pink color, add sidebar-pink class in #main-container's tag, like below: 

<!-- BEGIN Container -->
<div class="container sidebar-pink" id="main-container">
    <!-- BEGIN Sidebar -->
    <div id="sidebar" class="navbar-collapse collapse">
        ...
    </div>
    <!-- END Sidebar -->
    .
    .
    .

Consider that sidebar color, will overrides skin color.

Fixed Navbar & Sidebar

FLATY by default don't uses fixed navbar and sidebar. To enable this:

  1. Add navbar-fixed class to #navbar element <div id="navbar" class="navbar navbar-fixed">
  2. Add sidebar-fixed class to #sidebar element <div id="sidebar" class="navbar-collapse collapse sidebar-fixed">

Default Hidden Sidebar

To set default hidden sidebar just add sidebar-collapsed class to the #sidebar as shown below: 

<!-- BEGIN Sidebar -->
<div id="sidebar" class="navbar-collapse collapse sidebar-collapsed">
    ...
</div>
<!-- END Sidebar -->
